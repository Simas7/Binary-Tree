# Binary Search Tree in C

## 🇧🇷 Português

Este projeto implementa uma Árvore Binária de Busca (BST) em linguagem C com fins educacionais. Nele são abordados conceitos fundamentais de estruturas de dados, incluindo inserção de elementos, percursos recursivos (pré-ordem, em ordem e pós-ordem), busca pelos menores e maiores valores e visualização da estrutura da árvore no terminal. O objetivo é servir como material de apoio para estudantes e desenvolvedores que desejam compreender o funcionamento interno de árvores binárias e algoritmos recursivos.

## 🇺🇸 English

This project implements a Binary Search Tree (BST) in C for educational purposes. It covers fundamental data structure concepts, including node insertion, recursive traversals (pre-order, in-order, and post-order), searching for the minimum and maximum values, and visualizing the tree structure in the terminal. The main goal is to provide a learning resource for students and developers who want to better understand how binary trees and recursive algorithms work internally.
